# Contributing

## Commit Style
- Follow Conventional Commits (e.g., `feat: add drone route`, `fix: handle SSE reconnect`).

## PR Requirements
- CI green (typecheck, lint, tests).
- Keep PRs small and focused.
- Update docs where relevant.

## Setup
- pnpm i
- pnpm dev
- See `docs/runbooks/local-dev.md`.
